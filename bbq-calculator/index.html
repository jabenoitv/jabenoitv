📡 Sobre la sonda iGrill 2

Weber no ofrece una API pública para el iGrill 2, así que no es posible leer la sonda directamente desde el navegador. La forma práctica de usarla aquí es:

  1. Mira la temperatura en tu app oficial iGrill.
  2. Activa el modo "Sonda iGrill 2 (con recordatorio)" en el Monitor.
  3. La app te avisará cada X minutos para copiar la lectura.

Si tienes Home Assistant con integración iGrill, podrías automatizarlo a futuro.
